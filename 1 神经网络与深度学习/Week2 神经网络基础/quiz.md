#### 10 广播

```python
a = np.random.randn(3, 3)
b = np.random.randn(3, 1)
c = a * b
```

这将会使用广播机制，b会被复制三次，就会变成(3,3)，再使用元素乘法。所以： `c.shape = (3, 3)`
